# Archived Implementation Plans

Historical per-phase design and implementation notes from the initial build
(Feb 2026). Retained for context on why certain decisions were made. These
documents are **not** current — refer to `README.md` and `CLAUDE.md` in the
repo root for up-to-date architecture and usage.

Each `*-design.md` outlines the goal and approach for a phase; the matching
`*-implementation.md` records the concrete tasks executed in that phase.
